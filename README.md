# Yahdi Qalbah — Plateforme

Plateforme web de l'Institut **Yahdi Qalbah** — enseigner et faire aimer le Coran.

## Pièce n°1 — La Chaîne des Prophètes

**Live : https://chaine-des-prophetes.vercel.app**

Timeline interactive des 25 prophètes cités dans le Coran, de Adam à Muhammad ﷺ :

- Fil d'or déroulé au scroll, ciel étoilé, étoiles filantes
- 25 stations (nom arabe, époque, accroches) groupées en 4 ères
- **Un récit complet et lisible par prophète** (lecteur plein écran, navigation clavier)
- **100 % sourcé** : ~140 références coraniques cliquables vers quran.com (traduction FR)
- Fichier unique autonome (`index.html`), zéro dépendance, mobile + `prefers-reduced-motion`

## Stack

HTML/CSS/JS vanilla — Cormorant Garamond · Plus Jakarta Sans · Amiri. Déployé sur Vercel.
